Hypertext Transfer Protocol

Definitions

Hypertext Transfer Protocol [HTTP] Application layer (OSI layer 7) communication protocol created by Sir Tim Berners-Lee. Typically using TCP/IP transport and networking protocols for network functionality, and designed around the client server model.
Uniform Resource Locators [URL] A standardized resource address, typically used for resources on other machines. Created by Sir Tim Berners-Lee.
Open System Interconnection Model [OSI] A abstract model for differenciating different levels of abstraction of machine communications.
European Organization for Nuclear Research [CERN] A research organization, focusing on physics related research, located in Switzerland.
HTTP Request A HTTP message from the client
HTTP Response A HTTP message responding to the client, from the server
